# gmusic
a simple client for  [gmusicproxy](http://gmusicproxy.net/)

```
Usage:
  gmusic [command]

Available Commands:
  collection  get all the songs in your personal collection as an M3U playlist.
  disco       returns the list of available albums of the specified artist as an M3U playlist
  ifl         get the tracks associated to the automatic “I’m feeling lucky” station
  search      search for music
  top         returns an M3U playlist with the top songs of a specified artist

Flags:
      --addr string   the address of the gmusicproxy server (default "http://localhost:9999")
```
